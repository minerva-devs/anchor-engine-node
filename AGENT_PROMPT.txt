## Critical Constraints & Requirements
| Requirement | Why it matters | How to handle failures |
|-------------|----------------|------------------------|
| **No Unicode in logs** | CMD/PowerShell 5.1 can't handle emojis | Replace all emoji with `[OK]`, `[WARN]`, etc. |
| **ASCII-only file paths** | Windows terminals are sensitive to `\U` escapes | Use `r"..."` raw strings for paths |
| **Binary-safe write** | Avoid terminal corruption when creating files | Write using UTF-8 encoding, or Python's `.write()` |
| **Skip missing UI elements gracefully** | File creation/distillation buttons may not exist yet | Catch exceptions and log `[PAUSE]` instead of crashing |
